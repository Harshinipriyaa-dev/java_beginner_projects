
# Java Beginner Projects ☕

This repository contains my early object-oriented programming (OOP) projects in Java. It highlights my understanding of classes, file handling, and working with Java Streams.

## 📁 Projects Structure

### 🔒 Mini Password Locker
A terminal-based credential manager that securely stores and reads passwords using Java's object serialization.
* *Key Concept Used:* Subclassed ObjectOutputStream and overrode writeStreamHeader() to dynamically handle appending data without stream header corruption.

### 🕵️‍♂️ The Conspiracy Files
A custom project exploring file input/output manipulation and data management in Java.

## ⚙️ Tech Stack & Concepts
* *Language:* Java
* *Concepts:* OOPs, File I/O, Serialization, Custom Exceptions, Exception Handling
